# Dil Designs — Artisan Hand-Painted Dolls

A luxury e-commerce website showcasing hand-painted, one-of-a-kind artisan dolls by Danilo Reyes.

## 🎭 Overview

This is a production-grade, high-end website for Dil Designs. The site features:

- **12 unique artisan dolls** across genres (Victorian Gothic, Hipster, Cottagecore, Cyberpunk, Art Deco, Fairy Tale, Asian Inspired, Steampunk, Masquerade, Bohemian, Minimalist Modern, French Vintage)
- **Luxury aesthetic** with refined typography, negative space, and a sophisticated color palette
- **Interactive gallery** with modal lightbox for detailed doll viewing
- **Artist bio** featuring Danilo Reyes, the visionary creator
- **Mobile-responsive design** optimized for all devices

## 🚀 Quick Start

### Local Viewing

1. **Open in browser:**
   ```bash
   open ~/.openclaw/workspace/dil-dolls/index.html
   ```

2. **Or use a local server (Python 3):**
   ```bash
   cd ~/.openclaw/workspace/dil-dolls && python3 -m http.server 8000
   ```
   Then visit: `http://localhost:8000`

### Deploy to Netlify (Free)

1. **Install Netlify CLI:**
   ```bash
   npm install -g netlify-cli
   ```

2. **Deploy:**
   ```bash
   cd ~/.openclaw/workspace/dil-dolls
   netlify deploy --prod --dir .
   ```

3. **Get your live URL** and share it!

### Deploy to Vercel (Free)

1. **Push to GitHub:**
   ```bash
   cd ~/.openclaw/workspace/dil-dolls
   git init
   git add .
   git commit -m "Initial commit: Dil Designs website"
   git remote add origin https://github.com/YOUR_USERNAME/dil-designs.git
   git push -u origin main
   ```

2. **Connect to Vercel:** https://vercel.com/import

## 🎨 Design Details

### Color Palette
- **Cream:** `#f5f1e8` (background)
- **Charcoal:** `#2a2a2a` (text)
- **Gold:** `#d4a574` (accents)
- **Sage:** `#8b9e8d` (secondary text)
- **Dusty Rose:** `#a88b9e` (optional accents)

### Typography
- **Font:** Garamond / Georgia serif
- **Letter spacing:** Generous, elegant
- **Hierarchy:** Clean, minimalist

### Layout
- **Navigation:** Sticky header with collection/about/contact links
- **Hero:** Compelling headline + CTA
- **Gallery:** Responsive grid (3 columns on desktop, 1 on mobile)
- **About:** Artist bio with placeholder image
- **Contact:** Call-to-action section
- **Footer:** Simple copyright
